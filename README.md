THis project is a Checkers Game (Python + Pygame).

This project is a simple implementation of the classic Checkers (Draughts) board game. It is built using Python and Pygame library. It lets the player do basic gameplay like selecting pieces, normal moves, capturing moves, promoting pieces to king and switch turns.

Features:

A checkered color board consisting of 64 squares.
Two-player mode (Red vs Black).
Move pieces diagonally based on turn.
Capture opponent pieces by jumping.
When a piece reaches the opposite end, promote to king automatically.
Move highlighting system to show legal moves.
Capture grants an extra turn.
Highlighting selected pieces and movable squares.
Tech Stack:

Component Technology.
Language Python.
Graphics Pygame Library.
Installation & Setup:

Install Python: Ensure that Python 3.x is installed on your system. Check version using.
python --version.

Install Pygame. Run the following command in terminal / CMD.
pip install pygame.

Run the Game. You can save the script as checkers.py and run it.
python checkers.py.

How to Play: Action Instructions:

Select a piece Click on a red/black piece during its turn.
Move piece Click on a highlighted square.
Capture opponent Jump over opponent’s piece.
King promotion Reach the opposite last row.
